Selectores Texto Colores Box Model

Selectores Básicos y de Atributo

        
                /* Básicos */
                * { box-sizing: border-box; }
                h1 { font-size: clamp(1.5rem, 5vw, 2.5rem); }
                .destacado { font-weight: 700; }
                #encabezado { text-align: center; }

                /* Atributo */
                input[type="text"] { border: 1px solid #ccc; }
                a[href*="github"] { color: #333; } /* contiene 'github' */
                    
                /* Combinadores */
                article p { line-height: 1.6; }   /* descendiente */
                nav > ul { list-style: none; }    /* hijo directo */
                h2 + p { margin-top: 0; }         /* adyacente */

                /* Pseudo-clases y Elementos */
                a:hover { text-decoration: underline; }
                li:first-child { font-weight: 700; }
                ::selection { background: #ffda73; }

Tipografía y Formato de Texto

Define cómo se lee el contenido en tu sitio:


            /* Texto base optimizado */
            .texto {
            font-family: "Lato", system-ui, sans-serif;
            font-size: 1.1rem; /* Recomendado usar rem */
            font-weight: 400;
            line-height: 1.6;  /* Espaciado para mejor lectura */
            color: #282828;
            letter-spacing: 0.01em;
            }

            /* Títulos Responsive con clamp */
            .titulo {
            font-size: clamp(1.6rem, 4vw, 3rem);
            font-weight: 700;
            line-height: 1.1;
            }

            /* Truco: Cortar texto con puntos suspensivos (...) */
            .truncado {
            width: 220px;
            white-space: nowrap;
            overflow: hidden;
            text-overflow: ellipsis;
            }

Formatos y Estilos Visuales

                    
                    
Controla colores, fondos y efectos de profundidad:

                /* Colores y Sombras */
                .color  { color: #785082; }
                .fondo  { background: rgb(240, 180, 0); }
                .sombra { box-shadow: 0 8px 18px rgba(0,0,0,0.12); }

                /* Fondo con Imagen (Hero) */
                .hero {
                background-image: url("ruta/imagen.jpg");
                background-repeat: no-repeat;
                background-position: center;
                background-size: cover; /* Ajusta la imagen al contenedor */
                }

                /* Gradientes Modernos */
                .banner {
                background: linear-gradient(135deg, #785082, #6990aa);
                }

                /* Uso de Variables */
                .btn {
                color: var(--white);
                background: var(--secondary);
                }
